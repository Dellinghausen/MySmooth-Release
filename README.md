# MySmooth Release

This repository hosts the public distribution files for MySmooth.

It contains:

- The public website
- The remote `manifest.json`
- Public release downloads

The application source code is kept in a private repository. This repository is intentionally limited to files needed by end users and by the app update/configuration flow.

## Links

- Website: `https://dellinghausen.github.io/MySmooth-Release/`
- Manifest: `https://dellinghausen.github.io/MySmooth-Release/manifest.json`
- Downloads: `https://github.com/Dellinghausen/MySmooth-Release/releases/latest`

## Status

MySmooth is currently planned as a free beta with optional support.

MySmooth

A simple Path of Exile utility for visual cleanup, readability, backups, and restoration.

Download Support the project Discord

What it does

Visual

Map readability, atlas cleanup, modifier colors, and camera zoom.

Light performance

Lower-risk cleanup for rain, clouds, fog, shadows, lights, and environment effects.

Heavy performance

Optional aggressive cleanup that can hide skill effects. Use carefully.

Support model

MySmooth is planned as a free beta with optional contributions. Contributions help cover hosting, signing, testing time, and compatibility work after game updates.

Important

MySmooth is unofficial and not affiliated with Grinding Gear Games. It modifies local game data and should be used at your own risk. Always restore originals or verify game files if something looks wrong.
